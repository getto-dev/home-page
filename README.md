# Закладки на новой вкладке

Chrome-расширение для отображения закладок на странице новой вкладки.

## Возможности

- 📁 Папки и подпапки закладок
- 🔍 Сортировка (по умолчанию, по имени, по дате)
- 📱 Адаптивный дизайн (2/3/4 колонки)
- 🌙 Тёмная тема (автоматически)
- ⚡ Быстрый доступ к диспетчеру закладок

## Установка

1. Скачайте репозиторий (Code → Download ZIP)
2. Распакуйте архив
3. Откройте `chrome://extensions/`
4. Включите **Режим разработчика**
5. Нажмите **Загрузить распакованное расширение**
6. Выберите папку с расширением

## Использование

- **Левый клик** — открыть в текущей вкладке
- **Колёсико** — открыть в новой вкладке (фоном)
- **Правый клик** — контекстное меню
